Mwaki Denis Portfolio
A personal portfolio website showcasing the work and skills of Mwaki Denis, a software developer specializing in fullstack development, mobile applications, and cybersecurity.

Features
Hero Section: Introduction with personal details and enhanced "View Live Demo" button with gradient styling and animations
About Me: Overview of skills and experience
Tech Stack: Display of technologies and tools with interactive hover effects
Connect: Social media and contact links
GitHub Analytics: Stats, streaks, top languages, and activity graph
Contact Form: Direct messaging capability
Live Demo: Interactive demo of the portfolio with JavaScript animations and graphics
Particle Graphics: Dynamic background particle system that responds to mouse movement
Scroll Animations: Sections fade in with smooth upward motion when scrolled into view
Interactive Elements: Enhanced hover effects, click animations, and game-like interactions
Technologies Used
HTML5
CSS3 (Tailwind CSS)
JavaScript (Vanilla JS for animations and particle system)
Font Awesome Icons
GitHub Readme Stats API
Canvas API for particle graphics
